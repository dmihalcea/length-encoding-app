# Length Encoding App

A simple form of data compression where consecutive
repeated characters are replaced by the character followed by its count.

## Usage
1. Run `Main.java`.
2. Check the `Pre-Tests`
2. Add your `String`
3. Check the **final output**

## Input Examples
- `wwwwaaadexxxxxx`
- `aaaabbbccc`
- `abbbcdddd`
- `aabcccaazzzz`

## Output Examples
- `w4a3d1e1x6`
- `a4b3c3`
- `a1b3c1d4`
- `a2b1c3a2z4`
